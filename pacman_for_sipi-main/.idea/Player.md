# 🔵 Класс `Player`
📁 `pacman.py` | 🎮 Управление персонажем

## 📋 Назначение
Представляет персонажа Pac-Man, управляет его движением, состоянием и взаимодействием с игровыми объектами.

## 🧩 Атрибуты
| Атрибут | Тип | Описание |
|---------|-----|----------|
| `position` | `tuple` | Текущие координаты `(x, y)` |
| `direction` | `str` | Текущее направление (`'up'`/`'down'`/`'left'`/`'right'`) |
| `next_direction` | `str` | Запрошенное следующее направление |
| `lives` | `int` | Количество жизней (по умолчанию `3`) |
| `score` | `int` | Счёт игрока |
| `speed` | `float` | Скорость перемещения |
| `start_pos` | `tuple` | Стартовая позиция на уровне |

## ⚙️ Методы
| Метод | Описание |
|-------|----------|
| `__init__(start_pos, speed)` | Создание экземпляра игрока |
| `move(level)` | Перемещение с учётом стен и направления |
| `change_direction(new_dir)` | Изменение вектора движения |
| `eat(coin)` | Обработка поедания монеты |
| `die()` | Обработка гибели персонажа |
| `respawn()` | Возрождение на стартовой позиции |
| `check_collision(ghosts)` | Проверка столкновений с призраками |

## 🔗 Взаимодействие
- **Зависит от:** `Level`, `Coin`, `Ghost`
- **Вызывается из:** `Game.update()`, `controls.py`
