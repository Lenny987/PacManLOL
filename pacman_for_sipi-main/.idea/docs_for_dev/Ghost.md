# 👻 Класс `Ghost`
📁 `ghosts.py` | 🤖 AI противников

## 📋 Назначение
Описывает призрака, реализует алгоритмы искусственного интеллекта для навигации и управляет переходами между состояниями.

## 🧩 Атрибуты
| Атрибут | Тип | Описание |
|---------|-----|----------|
| `name` | `str` | Имя призрака (`Blinky`/`Pinky`/`Inky`/`Clyde`) |
| `position` | `tuple` | Текущие координаты |
| `direction` | `str` | Направление движения |
| `state` | `str` | Состояние (`'normal'`/`'frightened'`/`'dead'`) |
| `speed` | `float` | Скорость перемещения |
| `color` | `str/tuple` | Цвет отображения |
| `start_pos` | `tuple` | Стартовая позиция («домик») |
| `frightened_timer` | `int` | Таймер испуганного состояния |

## ⚙️ Методы
| Метод | Описание |
|-------|----------|
| `__init__(name, start_pos, color, speed)` | Создание призрака |
| `move(player_position, level)` | Навигация по алгоритму AI |
| `change_state(new_state)` | Смена режима поведения |
| `respawn()` | Возврат в домик |
| `update_frightened_timer()` | Обновление таймера испуга |
| `get_target_tile(player_pos)` | Расчёт целевой клетки для преследования |

## 🔗 Взаимодействие
- **Зависит от:** `Level`, `Player`
- **Управляется:** `Game.update()`, логикой силовых монет
