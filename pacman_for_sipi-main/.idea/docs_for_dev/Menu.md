# 🖥️ Класс `Menu`
📁 `start_game.py`, `game_over_menu.py` |  Навигация

## 📋 Назначение
Управляет системой меню и навигацией между экранами приложения. Отвечает за отрисовку UI и обработку ввода в неигровых состояниях.

## 🧩 Атрибуты
| Атрибут | Тип | Описание |
|---------|-----|----------|
| `current_screen` | `str` | Текущий экран (`main`/`settings`/`leaderboard`/`pause`/`game_over`) |
| `selected_item` | `int` | Выбранный пункт меню |
| `buttons` | `list` | Список кнопок текущего экрана |
| `input_text` | `str` | Вводимый текст (для имени игрока) |

## ⚙️ Методы
| Метод | Описание |
|-------|----------|
| `__init__()` | Создание экземпляра меню |
| `render(screen)` | Отрисовка текущего экрана |
| `handle_input(event)` | Обработка ввода пользователя |
| `navigate(direction)` | Навигация по пунктам меню |
| `show_main_menu()` | Отображение главного меню |
| `show_settings()` | Отображение настроек |
| `show_leaderboard(records)` | Отображение лидерборда |
| `show_pause()` | Отображение экрана паузы |
| `show_game_over(score)` | Отображение экрана завершения |

## 🔗 Взаимодействие
- **Зависит от:** `button.py`, `Settings`, `Leaderboard`
- **Управляется:** `Game.change_state()`
