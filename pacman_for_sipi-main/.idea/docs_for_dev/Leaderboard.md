# 🏆 Класс `Leaderboard`
📁 `leaderboard.py` | 📊 Таблица рекордов

## 📋 Назначение
Управляет таблицей лидеров, обеспечивает сохранение, загрузку и защиту данных рекордов.

## 🧩 Атрибуты
| Атрибут | Тип | Описание |
|---------|-----|----------|
| `records` | `list[dict]` | Список записей (`name`, `score`, `date`, `level`) |
| `max_records` | `int` | Максимальное число записей (`10`) |
| `file_path` | `str` | Путь к файлу `leaderboard.txt` |
| `checksum` | `int` | Контрольная сумма CRC32 |

## ⚙️ Методы
| Метод | Описание |
|-------|----------|
| `__init__(file_path)` | Инициализация лидерборда |
| `add_record(name, score, level_name)` | Добавление новой записи |
| `load()` | Загрузка из файла |
| `save()` | Сохранение в файл с расчётом CRC32 |
| `get_top_10()` | Получение топ-10 результатов |
| `verify_checksum()` | Проверка целостности данных |
| `sort_records()` | Сортировка по убыванию счёта |

## 🔗 Взаимодействие
- **Зависит от:** `txt`, `binascii` (CRC32)
- **Используется:** `Game`, `Menu.show_leaderboard()`
