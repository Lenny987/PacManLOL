# 🧱 Класс `Level`
📁 `field.py` | 🗺️ Карта и сетка уровня

## 📋 Назначение
Описывает игровой уровень (лабиринт), управляет двумерной сеткой клеток, хранением стен и монет.

## 🧩 Атрибуты
| Атрибут | Тип | Описание |
|---------|-----|----------|
| `grid` | `list[list]` | Двумерный массив клеток |
| `width` | `int` | Ширина уровня в клетках |
| `height` | `int` | Высота уровня в клетках |
| `initial_coins` | `int` | Начальное количество монет |
| `name` | `str` | Название уровня |
| `walls` | `list[tuple]` | Список координат стен |
| `coins` | `list[Coin]` | Список объектов монет |

## ⚙️ Методы
| Метод | Описание |
|-------|----------|
| `__init__(name, width, height)` | Создание пустого уровня |
| `load(filepath)` | Загрузка уровня из JSON-файла |
| `get_tile(x, y)` | Получение типа клетки |
| `is_wall(x, y)` | Проверка на стену |
| `count_coins()` | Подсчёт оставшихся монет |
| `save(filepath)` | Сохранение уровня в JSON |
| `validate()` | Проверка корректности структуры |

## 🔗 Взаимодействие
- **Зависит от:** `Coin`, `JSON`, `LevelEditor`
- **Используется:** `Player`, `Ghost`, `Game`
