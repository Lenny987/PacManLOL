# 🔊 Класс `SoundManager`
📁 `sound_manager.py` (или встроен в `main.py`) | 🎵 Аудио-система

## 📋 Назначение
Управляет воспроизведением звуков и фоновой музыки. Централизует работу с `pygame.mixer`.

## 🧩 Атрибуты
| Атрибут | Тип | Описание |
|---------|-----|----------|
| `volume` | `float` | Общая громкость |
| `sounds` | `dict` | Словарь звуковых объектов |
| `music_loaded` | `bool` | Флаг загрузки музыки |

## ⚙️ Методы
| Метод | Описание |
|-------|----------|
| `__init__()` | Инициализация микшера |
| `load_sounds()` | Загрузка звуковых файлов из `data/musics/` |
| `play(sound_name)` | Воспроизведение звука |
| `set_volume(value)` | Установка громкости |
| `play_music()` | Запуск фоновой музыки |
| `stop_music()` | Остановка музыки |
| `pause_music()` | Пауза музыки |

## 🔗 Взаимодействие
- **Зависит от:** `pygame.mixer`, `Settings`
- **Вызывается:** `Game`, `Player`, `Ghost`, `Level`
