# 🪙 Класс `Coin`
📁 `grain.py`, `super_grain.py` | 💎 Игровые объекты

## 📋 Назначение
Описывает монету (обычную или силовую). Отвечает за отображение, анимацию и логику сбора.

## 🧩 Атрибуты
| Атрибут | Тип | Описание |
|---------|-----|----------|
| `position` | `tuple` | Координаты монеты |
| `value` | `int` | Ценность в очках (`10` / `50`) |
| `is_power` | `bool` | Признак силовой монеты |
| `active` | `bool` | Активность (не собрана ли) |
| `animation_frame` | `int` | Текущий кадр анимации |

## ⚙️ Методы
| Метод | Описание |
|-------|----------|
| `__init__(position, value, is_power)` | Создание монеты |
| `collect()` | Сбор монеты (`active = False`) |
| `draw(screen)` | Отрисовка на экране |
| `update()` | Обновление кадра анимации |

## 🔗 Взаимодействие
- **Зависит от:** `pygame`, `Level.grid`
- **Вызывается:** `Player.eat()`, `Level.load()`
