📖 ПОЛНАЯ ИНСТРУКЦИЯ:
   Откройте файл USER_MANUAL.md в этой папке

🚀 БЫСТРЫЙ СТАРТ:
   1. Запустите main.exe
   2. Нажмите "Играть"
   3. Управляйте стрелками или WASD
   4. Соберите все монеты!

🎮 УПРАВЛЕНИЕ:
   ↑ ↓ ← →  или  W A S D  — движение
   ESC — пауза

⚙️ НАСТРОЙКИ:
   В главном меню нажмите "Настройки"
   - Громкость: 0-100%
   - Призраков: 1-4 штуки

🏆 ТАБЛИЦА ЛИДЕРОВ:
   В главном меню нажмите "Лидерборд"

🛠️ РЕДАКТОР УРОВНЕЙ:
   В главном меню нажмите "Редактор уровней"
   Создавайте свои лабиринты!

📁 СТРУКТУРА ПАПОК:
   data/           — игровые ресурсы
   levels/         — ваши уровни
   settings.txt    — настройки
   leaderboard.txt — таблица рекордов
