# 🛠️ Класс `LevelEditor`
📁 `map_editor_start.py` | 🧩 Конструктор уровней

## 📋 Назначение
Реализует редактор уровней для создания пользовательских карт. Позволяет размещать стены, монеты, позиции персонажей и валидировать структуру.

## 🧩 Атрибуты
| Атрибут | Тип | Описание |
|---------|-----|----------|
| `grid` | `list[list]` | Сетка уровня |
| `selected_tool` | `int` | Выбранный инструмент (`0`=стена, `1`=монета, `2`=силовая, `3`=пакоман, `4`=призрак) |
| `cursor_pos` | `tuple` | Позиция курсора на сетке |
| `level_name` | `str` | Название создаваемого уровня |

## ⚙️ Методы
| Метод | Описание |
|-------|----------|
| `__init__(grid_size)` | Инициализация редактора |
| `place_tile(x, y, tile_type)` | Размещение объекта |
| `remove_tile(x, y)` | Удаление объекта |
| `save(filepath)` | Сохранение уровня в JSON |
| `load(filepath)` | Загрузка уровня из файла |
| `validate()` | Проверка корректности карты |
| `clear()` | Очистка поля |
| `draw(screen)` | Отрисовка интерфейса редактора |

## 🔗 Взаимодействие
- **Зависит от:** `pygame.mouse`, `json`, `Level`
- **Вызывается:** `Menu.show_editor()`, `Game`
