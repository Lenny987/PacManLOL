# ⚙️ Класс `Settings`
📁 `settings.py` | 🔧 Конфигурация

## 📋 Назначение
Хранит и управляет пользовательскими настройками. Обеспечивает персистентность параметров между сессиями.

## 🧩 Атрибуты
| Атрибут | Тип | Описание |
|---------|-----|----------|
| `volume` | `float` | Громкость (`0.0`–`1.0`) |
| `ghost_count` | `int` | Количество призраков (`1`–`4`) |
| `first_launch` | `bool` | Флаг первого запуска |
| `resolution` | `tuple` | Разрешение экрана |
| `file_path` | `str` | Путь к `settings.json` |

## ⚙️ Методы
| Метод | Описание |
|-------|----------|
| `__init__(file_path)` | Инициализация настроек |
| `load()` | Загрузка из файла |
| `save()` | Сохранение в файл |
| `reset_defaults()` | Сброс к значениям по умолчанию |
| `set_volume(value)` | Установка громкости |
| `set_ghost_count(value)` | Установка числа призраков |

## 🔗 Взаимодействие
- **Зависит от:** `json`, `SoundManager`
- **Вызывается:** `Game.__init__()`, `Menu.show_settings()`
